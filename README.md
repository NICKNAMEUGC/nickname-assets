# Nickname Assets

Repositorio público de assets estáticos para Nickname Agency.

## Uso

**URL directa (GitHub Raw):**
```
https://raw.githubusercontent.com/NICKNAMEUGC/nickname-assets/main/{path}
```

**URL CDN (jsDelivr — cacheado globalmente, recomendado para emails):**
```
https://cdn.jsdelivr.net/gh/NICKNAMEUGC/nickname-assets@main/{path}
```

## Estructura

```
emails/           → Hero images y assets de email marketing
  welcome/        → Secuencia de bienvenida
  newsletter/     → Newsletter semanal
  winback/        → Reactivación
  retention/      → Fidelización
  abm/            → Prospección
  postcampaign/   → Post-campaña
linkedin/         → Assets de LinkedIn
  posts/          → Imágenes de posts
  logos/           → Logos para publicaciones
brand/            → Logos, favicon, brand assets
presentations/    → Assets de presentaciones
```

## Notas

- Todas las imágenes optimizadas para email (max 800px ancho, <100KB)
- Repo público — no subir datos sensibles
- Usado por: email-marketing-agent, n8n workflows, LinkedIn pipeline
